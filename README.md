# Farmer Crop Predictor

A machine learning web application that predicts the most suitable crop based on soil and environmental conditions.

# Features

- Crop prediction using Decision Tree Classifier
- User-friendly web interface
- Cost estimation
- Profit estimation

# Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- HTML
- CSS

# Demo Video

[Watch Demo on YouTube](https://youtu.be/z-CKW1FL1Xw)

# Input Parameters

- Nitrogen
- Phosphorus
- Potassium
- Temperature
- Humidity
- Rainfall
- Area
- Workers

# Installation

git clone https://github.com/tanbir-hsain/crop-prediction.git
cd crop-prediction
pip install -r requirements.txt
python app.py
